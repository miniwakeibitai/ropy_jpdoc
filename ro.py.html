    概要¶

    ro.py へようこそ ! ro.py は、Roblox Web API用の非同期のオブジェクト指向ラッパーです。

    顔立ち¶

    主な機能は次のとおりです。

    • 非同期: ro.py は、FastAPI や discord.py などの非同期フレームワークでうまく機能します。
    • 簡単:ro.pyのクライアントベースのモデルは、初心者とエキスパートの両方の開発者にとって直感的で簡単に習得できます。それ APIリクエストを抽象化し、Robloxプラットフォーム上のデータ型を表す単純なオブジェクトを残します。
    • 柔軟性:ro.pyの組み込みのRequestsオブジェクトを使用すると、ユーザーはまだ実装していないことを実行できます 高度なRoblox固有の概念を扱うことなく、私たち自身。

    取り付け¶

    PyPIから ro.py をインストールするには、pipを使用してインストールできます。

    pip install roblox

    git から ro.py をインストールするには、git-scm をインストールし、次のコマンドを実行します。

    pip install git+git://github.com/ro-py/ro.py.git
